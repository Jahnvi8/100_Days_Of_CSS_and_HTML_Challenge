The HTML tag defines a hyperlink. It has the following syntax: link text The most important attribute of the element is the href attribute, which indicates the link's destination. The link text is the part that will be visible to the reader. Clicking on the link text, will send the reader to the specified URL address. The target Attribute: By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link. The target attribute specifies where to open the linked document. The target attribute can have one of the following values: 1) _self - Default. Opens the document in the same window/tab as it was clicked 2) _blank - Opens the document in a new window or tab 3) _parent - Opens the document in the parent frame 4) _top - Opens the document in the full body of the window _______________________________________________________________________________ Visit W3Schools!
